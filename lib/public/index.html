Cats

This API uses a list of plant biomes and climbing towns as it's recourse. The routes are as follows:
Route Method Description Example 
/plants
 GET Gets a list of plant biomes and ids List Plant Biomes 
/plants/:id
 GET Returns the detail about the plant biome matching the id in the parameter Get Plant Biome 1 
/towns
 GET Gets a list of climbing towns and ids List climbing towns 
/towns/:id
 GET Returns the detail about the climbing town matching the id in the parameter Get town 1
